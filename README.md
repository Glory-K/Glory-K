# 김영광 | Kim YeongGwang

데이터 수집, 검증, 저장, 자동화, 서비스 연동까지 이어지는 흐름을 설계하고 구현하는 데 관심이 있습니다.  
형식이 다른 데이터를 활용 가능한 구조로 정리하고, 분석 및 서비스에 연결하는 프로젝트를 수행해왔습니다.

## 주요 분야
- Python / Selenium 기반 웹 데이터 수집 및 검증
- PostgreSQL / dbt 기반 데이터 모델링 및 계층형 구조 설계
- Django 기반 백엔드 및 API 연동
- 공공데이터 / 외부데이터 구조화 및 분석 기준 설계

## 대표 프로젝트

### E-commerce Data Platform
- PostgreSQL과 dbt를 활용한 배치형 데이터 플랫폼 구축
- source → staging → intermediate → marts 계층 설계
- 일별 매출 / 월별 코호트 리텐션 마트 구현  
[Repository](https://github.com/Glory-K/de-project/tree/main/ecommerce-data-platform)

### Daegu Crime Analysis
- 형식이 다른 공공데이터를 자치구 단위로 표준화
- 범죄율, CCTV 밀도, 상업지역 비율 등 비교 지표 설계
- 단순 건수 비교가 아닌 해석 가능한 분석 구조 구성  
[Repository](https://github.com/Glory-K/Daegu_Crime_Analysis)

### safe-travel-app-backend
- 외부 안전 데이터를 서비스 목적에 맞게 구조화
- 백엔드, DB 설계, RESTful API 개발 수행
- 이질적인 데이터를 사용자 기능과 연결  
[Repository](https://github.com/DOOINSEO/safe-travel-app-backend)

### BLC
- 이미지 인식 결과를 결제·재고 흐름으로 연결
- 백엔드 및 DB 연동 구현
- 팀장으로서 협업 조율 수행  
[Repository](https://github.com/HyojungKim2022/Final-project)

## 기술 스택
Python, SQL, PostgreSQL, MySQL, dbt, Django, Selenium, Pandas, Docker, GitHub
